# element_vue3.0/VE-Admin

## 项目简介

基于**vue3.0**和**element-plus**开发的企业后台管理模板.

## 功能特性

项目使用了最新的**vue3.0全家桶**+**element-plus**+**mockjs**+**axios**.项目继成了**mockServe**,可脱离后端自主开发测试
对**axios**深度封装,采用动态路由,路由配置更简单,**mockServe**独立开发测试时在nodework直观查看接口数据



## 环境依赖

**node 14+**, **vueCli 4+**

## 部署步骤

npm i  
npm run serve
